# App Store Screenshots Guide

## Required Screenshot Sizes

| Device | Size (px) | Required |
|--------|-----------|----------|
| iPhone 6.7" (14 Pro Max) | 1290 x 2796 | Yes |
| iPhone 6.5" (11 Pro Max) | 1284 x 2778 | Yes |
| iPhone 5.5" (8 Plus) | 1242 x 2208 | Yes |

## Recommended Screenshots (5-10 per size)

1. **Dashboard** - Ana sayfa, bakiye ve istatistikler
2. **Create Shipment** - Yeni gönderi oluşturma ekranı
3. **Shipment List** - Gönderi listesi ve filtreleme
4. **Tracking** - Gerçek zamanlı takip ekranı
5. **Price Calculator** - Fiyat hesaplama
6. **Balance** - Bakiye yönetimi

## How to Capture Screenshots

### Option 1: Xcode Simulator
1. Open Xcode
2. Run app on iPhone 14 Pro Max simulator
3. Press Cmd + S to save screenshot
4. Repeat for other devices

### Option 2: Real Device
1. Connect iPhone to Mac
2. Open app and navigate to desired screen
3. Press Side Button + Volume Up
4. Use Preview to resize if needed

### Option 3: Figma/Sketch Mockups
1. Use device frame templates
2. Add app screenshots inside frames
3. Export at required sizes
